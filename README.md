# Russian VPS Web Panel (Веб-панель для VPS)

Это веб-приложение предоставляет эмуляцию терминала и мониторинг ресурсов сервера с интерфейсом на русском языке. Оно упаковано в Docker и готово к деплою.

## Функционал
- **Веб-терминал**: Полноценный доступ к bash/shell через браузер.
- **Мониторинг**: Отображение CPU, RAM и времени работы в реальном времени.
- **Русский интерфейс**: Все элементы UI локализованы.

## Быстрый запуск (Docker)

```bash
docker build -t vps-panel .
docker run -p 3000:3000 -it vps-panel
```

Затем откройте `http://localhost:3000` в браузере.

## Развертывание (Workflow)
Проект содержит `.github/workflows/deploy.yml` для автоматической сборки образа.

### ⚠️ Предупреждение о безопасности
Этот проект предоставляет **root доступ** (или доступ пользователя контейнера) через веб-интерфейс. 
1. Не разворачивайте его в открытом интернете без добавления слоя аутентификации (пароль/OAuth).
2. Использование GitHub Actions в качестве хостинга (майнинг, длительные сервера) запрещено правилами GitHub.
